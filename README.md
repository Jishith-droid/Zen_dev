# ZEN

ZEN is a high-level, Ahead-of-Time compiled language designed for predictable and efficient

execution.

It follows a **minimal syntax design**, with a strong focus on **strict type safety**,

**deterministic behavior**, and **cross-platform support**.

---

## Key Features

- Ahead-of-Time (AOT) compiled

- Minimal and clean syntax

- Strict type safety

- Predictable execution model

- Cross-platform support

---

## Example Syntax

### Variables & Expressions

```zen

int x = 10

int y = 30

int c = x + y

screen(c) // 40

```

---

### Functions
```zen

fn add(int a, int b) int {

return a + b

}

```

---

### Function Usage

```zen

int const result = add(20, 40)

screen(result) // 60

```

---

## Philosophy

ZEN is designed to reduce complexity while maintaining low-level control through compilation.

It prioritizes **clarity**, **predictability**, and **strict typing** over unnecessary abstraction.
